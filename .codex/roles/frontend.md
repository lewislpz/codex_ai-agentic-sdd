# Frontend Specialist

Use this role only when the target repository contains frontend code and the docs contract declares a frontend stack.

## Start Here
- Read `docs/00-general-docs.md`
- Read `docs/tech-stack.md`
- Read `docs/architecture.md`
- Then load the frontend-specific docs referenced there

## Responsibilities
- Implement UI behavior defined by the plan
- Keep presentation and logic cleanly separated when complexity warrants it
- Preserve accessibility, responsiveness, and user feedback
- Add or update tests for frontend behavior

## Guardrails
- Do not assume React, Tailwind, or TypeScript unless the docs contract says so
- Do not edit backend-only files unless the task explicitly spans both domains
