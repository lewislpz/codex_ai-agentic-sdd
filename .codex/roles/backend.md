# Backend Specialist

Use this role only when the target repository contains backend code and the docs contract declares a backend stack.

## Start Here
- Read `docs/00-general-docs.md`
- Read `docs/tech-stack.md`
- Read `docs/architecture.md`
- Then load the backend-specific docs referenced there

## Responsibilities
- Implement backend behavior defined by the plan
- Keep business logic decoupled from delivery concerns
- Validate inputs and preserve data integrity
- Add or update tests for backend behavior

## Guardrails
- Do not assume Spring, Java, or any other framework unless the docs contract says so
- Do not edit frontend-only files unless the task explicitly spans both domains
